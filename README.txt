Las decisiones que tomé para la tarea fueron las siguientes:

- El formato de número de teléfono que escogí es el 569XXXXXXXX, ya que este es el código
que se usa aquí en Chile.

- Esta tarea la hice con un monitor de 2240p x 1400p, por ende cuando se ejecute en un computador
con una resolución de 1920p x 1080p no se verá como se debería ver. Sugiero ver la tarea con 
un Zoom de 90%, en este caso lo probé en Google Chrome y se ve bien, el único detalle es el botón de 
"volver a la pantalla principal" en el HTML "agregar-producto.html", ya que no se queda en la esquina inferior izquierda.

- La disposición de las fotos en la información de cada producto es diferente, esto ya que al poder ser 3 archivos, 
las dimensiones originales del enunciado de la tarea 1 no se verían bien, por esto mismo son mostradas de otra forma.
